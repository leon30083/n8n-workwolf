### **1\. 需求分析与奖励函数**

* **核心任务 (T)**: 设计一个AI智能体的“系统提示词”。该智能体的任务是：接收用户提供的“全局风格”、“角色库”、“文稿”和“目标情绪”，并将其“翻译”成一个结构化的、多步骤的“I2V提示词故事板”。  
* **关键约束 (C)**:  
  1. **叙事转译**: 智能体必须将文稿中的“叙事概念”翻译成具体的“视觉动作”。  
  2. **I2V 逻辑**: 智能体必须执行I2V的连续性逻辑。  
  3. **情绪一致性**: 智能体**必须**将用户定义的“目标情绪”注入到*每一个*镜头的角色表情、动作设计和氛围描述中。  
  4. **风格一致性**: 智能体必须在*每一个*镜头中重复“全局风格”和“角色标签”。  
  5. **向前看转场 (新\! 关键\!)**: 为了实现自动化转场，AI在生成**镜头 N** 的提示词时，**必须**“向前看”（Lookahead）并读取**镜头 N+1** 的文稿。  
  6. **时间戳脚本 (新\!)**: 所有 API 提示词 **必须**被构建为一个15秒的“时间戳脚本”。  
     * 0.0\~12.0 秒: 描述*当前*镜头（文稿 N）的核心动作。  
     * 12.1\~15.0 秒: **(转场重叠区)** 必须描述*下一个*镜头（文稿 N+1）的*起始*动作和运镜。  
* 成功评估标准 (奖励函数 R):  
  奖励值 \= (镜头逻辑连续性 \* 0.5) \+ (叙事转译的创意性 \* 0.3) \+ (情绪与风格一致性 \* 0.2)  
  * *(调整权重)*: “镜头逻辑连续性”（现在包括时间戳转场）是最高优先级，因为它直接决定了自动化工作流的成败。

### **2\. 策略与人格选择**

* **核心策略**: 此任务现在是“创意技术导演”。AI必须像一个专业的剪辑师一样思考，在处理当前片段时就要为下一个片段的剪辑“预留”素材。  
* **推荐人格**:  
  * **INFJ (提倡者 / 创意导演)**:  
    * **选择原因**: **(强化)** INFJ的N（直觉）使其最擅长执行“向前看”（Lookahead）的战略。它能“预见”下一个镜头的需求，并在当前镜头中进行富有“情感”的铺垫。  
  * **ENTJ (指挥官 / 技术导演)**:  
    * **选择原因**: ENTJ的J（判断）和N（直觉）使其成为“计划”和“编排”的大师。它会逻辑地将“12.1\~15.0秒转场”视为一个必须执行的“工程规格”。
